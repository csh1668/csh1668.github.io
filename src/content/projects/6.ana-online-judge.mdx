- 동아리 프로그래밍 대회 개최를 위해 개발한 플랫폼
  - ANIGMA, ANAGETDON 등의 실제 동아리 대회 운영 사례 있음
- 일반, 스페셜저지, 인터렉티브 등 다양한 문제 형식과 8개 이상의 프로그래밍 언어 지원
- Next.js 풀스택에 채점 서버는 안정성과 속도를 위해 Rust + Redis 사용
- AWS 배포를 염두하여 S3 인터페이스와 호환되는 MinIO를 파일 저장소로 사용
- DMOJ처럼 SSH 등으로 실제 서버에 접속할 필요없이, 관리자 페이지 혹은 CLI로 모든 관리 작업을 수행할 수 있음
  - 사이트 테스트 시, LLM으로 테스트할 수 있도록 비즈니스 로직을 인증 API로 노출 및 CLI 개발: "AOJ에 A+B 문제 만들고 테스트해줘" -> LLM이 알아서 문제 지문과 테스트케이스까지 업로드하고 코드까지 제출 가능
