해외 영업 AI 자동화 에이전트 Rinda의 핵심 기능 중 하나인 "바이어 서치"의 문제점을 스스로 파악하고 정확도 및 성능을 개선

- **정확도 문제**: 기존 검색은 LLM 키워드 생성 기반 SQL 조회 방식이라, 사용자의 제품을 *판매할* 바이어가 아닌 *생산하는* 회사가 매칭되는 한계
- **속도 문제**: 외부 API 요청이 순차적으로 발생하며 3분 이상 소요
- **해결**: 총 5단계의 파이프라인으로 구성 (페르소나 추출 -> 멀티 소스 병렬 검색 -> 이메일 수집 -> 점수 기반 필터링 -> LLM 설명 추가)
- **결과**: 개선 후, 바이어 서치 품질에 대한 긍정적 고객 피드백 11건 확인. 기존 대비 비현실적 추천 문제 해소. 소요 시간을 최대 100초로 단축.
